WATERMELON SUGAR HIGH! Aprenda uma farm muito eficiente de frutos no Minecraft!

Este guia ensina como construir um mecanismo voador eficiente para colheita de melancias e abóboras no Minecraft. Ideal para jogadores que buscam automação e produtividade.

Por The Master Miner 17.10.2024 às 10:36

    ㅤ

  1. Construindo o Mecanismo Voador

  2. Este mecanismo utiliza pistões pegajosos, observadores e blocos de slime para criar um sistema voador de colheita automática. Siga as etapas abaixo para construí-lo.

  3. ㅤ

  4. Lista de Materiais

  6. Blocos de terra

  7. Blocos de slime

  8. Vagões de mina

  9. Blocos de obsidiana

  10. Pistões pegajosos

  11. Observadores

  12. Vidro

  13. Repetidores de redstone

  14. Portas-armadilha

  15. Passo a Passo

  16. 1. Base do Mecanismo

  17. Posicione dois blocos de terra e coloque um bloco de slime sobre cada um. Remova os blocos de terra e substitua-os por quatro blocos de slime. Adicione barras de ferro embaixo dos blocos de slime e posicione trilhos e vagões de mina. Remova os trilhos após posicionar os vagões.

  18. 2. Pistões e Observadores

  19. Construa colunas de dois blocos de altura para posicionar pistões pegajosos sob cada coluna. Conecte os pistões com blocos de terra e coloque observadores de frente para os pistões.

  20. 3. Estrutura de Vidro e Redstone

  21. Adicione vidro atrás dos vagões para bloqueá-los. Conecte os pistões e observadores com obsidiana e repetidores configurados para dois ticks. Crie um caminho de vidro em linha reta, alternando blocos para guiar o mecanismo voador.

  22. 4. Ativação

  23. Coloque uma porta-armadilha em cima do observador central para ativar o mecanismo. Repita o mesmo procedimento do outro lado para completar o sistema.

  24. Funcionamento

  25. O mecanismo voador se move automaticamente sobre a plantação de melancias e abóboras, quebrando os blocos e permitindo uma colheita contínua.

  26. Comentarios Finais

  27. Esta farm utiliza uma abordagem inovadora com mecanismos voadores para maximizar a produtividade. Certifique-se de seguir o guia corretamente para evitar erros na construção.
