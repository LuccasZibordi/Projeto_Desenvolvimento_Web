FEEL THE FIRE FROM BARBECUE! Construa uma farm de carne de porco assada!

Este guia ensina como construir uma farm de hoglins eficiente no teto do Nether, capaz de produzir mais de 3.500 costeletas de porco cozidas e 300 couros por hora. Ideal para jogadores da versão Java Edition do Minecraft.

Por Eyecraftmc 17.10.2024 às 10:36

    ㅤ

  1. Construindo uma Farm de Carne de Porco

  2. Esta farm utiliza mecânicas do Nether e o comportamento dos hoglins para criar uma fonte eficiente de costeletas de porco e couro. Siga as etapas abaixo para construí-la corretamente.

  3. ㅤ

  4. Lista de Materiais

  6. Lajes de pedra

  7. Vidro

  8. Blocos de verruga do Nether

  9. Alçapões de carvalho

  10. Fungos distorcidos

  11. Terra

  12. Placas

  13. Balde de lava

  14. Fogueiras de alma (opcional)

  15. Funis

  16. Baús

  17. Andaimes

  18. Passo a Passo

  19. 1. Escolha do Local

  20. Construa a farm no teto do Nether em uma Floresta Crimson. Utilize pérolas do fim para alcançar o nível Y=127. Instale um portal para facilitar o acesso ao local.

  21. 2. Construção da Câmara Mortífera

  22. Monte duas fileiras de baús duplos e conecte-os com funis para coleta dos itens. Utilize vidro para criar uma gaiola de 3 blocos de altura ao redor da área de coleta.

  23. Escolha o método de abate:

    • Costeletas cozidas: Use lava posicionada acima de placas para cozinhar os itens.
    • Costeletas cruas: Use fogueiras de alma sobre os funis para coletar os itens crus.

  24. 3. Construção das Plataformas de Spawn

  25. Construa cinco plataformas de lajes de pedra, cada uma com 7x8 blocos e espaçadas 2,5 blocos verticalmente. Adicione terra com fungos distorcidos no centro de cada plataforma, cercados por tochas para impedir o spawn de piglins.

  26. 4. Direcionamento dos Hoglins

  27. Utilize blocos de verruga do Nether cobertos com alçapões para atrair os hoglins em direção à câmara mortífera. Os hoglins tentarão alcançar os blocos e cairão na lava.

  28. 5. Toques Finais

  29. Construa uma plataforma AFK no topo da farm utilizando andaimes. Recoloque a lava na câmara mortífera e verifique se todas as plataformas estão funcionando corretamente.

  30. Funcionamento

  31. Os hoglins spawnam nas plataformas e são assustados pelos fungos distorcidos, correndo em direção à câmara de abate. Os itens são coletados pelos funis e armazenados nos baús.

  32. Comentarios Finais

  33. Esta farm é altamente eficiente, oferecendo uma fonte constante de alimento e couro. Certifique-se de seguir o guia cuidadosamente para otimizar os resultados.
