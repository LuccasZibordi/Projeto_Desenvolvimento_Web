VISH O TIGRINHO NAO PAGOU?! Crie uma farm das moedas villager de uma forma simples

Este guia ensina como construir uma farm de esmeraldas eficiente no Minecraft, utilizando troca de barbantes com aldeões flecheiros para maximizar a produção.

Por RealJayden 17.10.2024 às 10:36

    ㅤ

  1. Construindo a Farm de Esmeraldas

  2. Esta farm utiliza mecanismos simples de armadilha e troca com aldeões para gerar esmeraldas automaticamente. Siga os passos abaixo para construí-la.

  3. ㅤ

  4. Materiais Necessários

  6. Blocos de construção

  7. Lajes

  8. Vidro

  9. Baús e Funis

  10. Fio-armadilha

  11. Barbante

  12. Alavancas

  13. Balde de água

  14. Passo a Passo

  15. 1. Construção da Estrutura

  16. Cave um buraco de 3x4 no chão e posicione baús duplos conectados por funis. Construa uma parede de três blocos de altura ao redor da área e adicione uma camada superior de blocos para finalizar a estrutura.

  17. 2. Configuração das Armadilhas

  18. Coloque blocos temporários no topo da estrutura para criar uma plataforma suspensa. Adicione fio-armadilha com barbantes no segundo bloco acima dos funis e teste o sistema com tesouras para garantir que o barbante permanece intacto.

  19. 3. Sistema de Água

  20. Posicione baldes de água no lado oposto às armadilhas para direcionar os mobs para o sistema de coleta sem tocar no barbante. Certifique-se de que a água flui corretamente.

  21. 4. Ativação e Decoração

  22. Adicione uma alavanca acima das armadilhas e ative-a para abrir o sistema. Decore a estrutura com toras de madeira nas laterais do vidro e lajes no topo.

  23. 5. Troca de Barbantes por Esmeraldas

  24. Encontre aldeões flecheiros e evolua-os para o nível 3 (Mestre) para desbloquear a troca de barbante por esmeraldas. Reproduza os aldeões, se necessário, para maximizar a produção.

  25. Funcionamento

  26. Os mobs são direcionados para o sistema de coleta, gerando barbantes automaticamente. Troque os barbantes com aldeões flecheiros para obter esmeraldas.

  27. Comentarios Finais

  28. Esta farm oferece uma solução prática e eficiente para obtenção de esmeraldas. Certifique-se de configurar corretamente o sistema de armadilhas e mantenha aldeões suficientes para troca.
