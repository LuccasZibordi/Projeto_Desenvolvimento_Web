QUEM COM FERRO FARMA COM FERRO VAI SER FARMADO! Aprenda uma farm de ferro infinita!

Este guia detalhado ensina como construir uma farm de ferro eficiente no Minecraft, compatível com as versões Java e Bedrock. Utilize um design otimizado para maximizar a produção de golems de ferro!

Por Willzy 17.10.2024 às 10:36

    ㅤ

  1. Construindo uma Farm de Ferro

  2. A Farm de Ferro Média é projetada para gerar ferro de forma infinita e otimizada. Este guia cobre todas as etapas necessárias, desde a escolha do local até os detalhes do funcionamento.

  3. ㅤ

  4. Lista de materiais

  6. Blocos de Pedra (grande quantidade)

  7. Bancadas de Trabalho (40 unidades)

  8. Camas (40 unidades)

  9. Blocos de Vidro ou Folhas (para paredes e teto)

  10. Lajes (para teto e estruturas)

  11. Funil (1 unidade)

  12. Baú Duplo (1 unidade)

  13. Fogueiras (múltiplas)

  14. Balde de Água (múltiplos)

  15. Placa (1 unidade)

  16. Etiqueta (1 unidade, Java Edition)

  17. Trilhos e Carrinho (para sistema de medo, Java Edition)

  18. Aldeões Adultos (40 no total)

  19. Passo a Passo

  20. 1. Localização e Plataforma

  21. Construa a farm a pelo menos 50 blocos de altura ou sobre o oceano. Monte uma plataforma extensa de pedra com paredes ao redor. Utilize vidro ou folhas para evitar spawns indesejados.

  22. 2. Sistema de Água

  23. Posicione fontes de água nas laterais e centro da plataforma para criar correntes que empurrem os golems para o centro.

  24. 3. Cabines dos Aldeões

  25. Monte 4 cabines idênticas na plataforma principal, cada uma com 10 camas e bancadas de trabalho. Utilize lajes ou vidro no teto para proteção e evite blocos sólidos para prevenir spawns.

  26. Adicione 10 aldeões adultos a cada cabine, garantindo que estejam atribuídos às camas e bancadas.

  27. 4. Sistema de Coleta

  28. Monte um tubo de queda no centro da plataforma principal. Utilize fogueiras no fundo do tubo e funis conectados a baús para coleta de ferro.

  29. 5. Sistema de Medo (Java Edition)

  30. Instale trilhos ao redor da plataforma com um zumbi em carrinho, movendo-se continuamente para assustar os aldeões e aumentar a geração de golems.

  31. Funcionamento

  32. Os golems de ferro são gerados nas plataformas, empurrados pelas correntes de água para o tubo, queimados nas fogueiras, e o ferro é coletado automaticamente nos baús.

  33. Considerações Finais

  34. A Farm de Ferro Média oferece uma solução eficiente para recursos automáticos. Siga os passos cuidadosamente para garantir a funcionalidade e aproveite os benefícios!
